# 🚀 Resume Analyzer

A full-stack web app that analyzes how well a resume matches a job description.

## 🔥 Features

* Upload resume (PDF)
* Paste job description
* Get match percentage
* See missing skills
* Get improvement suggestions

## 🛠️ Tech Stack

* Frontend: React.js
* Backend: Node.js, Express.js
* PDF Parsing: pdf-parse
* Deployment: Netlify + Render

## 🌐 Live Links

* Frontend: https://extraordinary-cheesecake-fe0ad2.netlify.app
* Backend: https://resume-analyzer-sv7i.onrender.com

## ⚙️ Setup

### Backend

```bash
cd backend
npm install
node server.js
```

### Frontend

```bash
cd frontend/my-app
npm install
npm start
```
